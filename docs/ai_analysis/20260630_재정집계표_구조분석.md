## 📌 TL;DR
- 외부 시트는 4탭: 헌금 / 지출 / 재정집계표 / 수입재정집계표 (서비스계정에 공유받아 읽음)
- **수입재정집계표**는 앱이 가진 헌금 종류만으로 구현 가능 → 완료 (`/finance/income`)
- **재정집계표**(예산 대비 실적)는 앱의 지출 종류에 **관리비 대분류 항목들이 빠져 있어** 정확히 못 만듦 → 지출 종류 정비 선행 필요

## 재정집계표 (예산 대비 실적) — 컬럼
`B:관(대분류) | C:항목 | D:예산 | E:직전누계 | F:현누계 | G:총합계`
- E(직전누계)=입력값(현재 전부 `-`), D(예산)=입력값
- **F(현누계) = G(총합계) = 실제 누계** (앱 데이터로 계산)

### 수입 구조
- 정기헌금: 주정 / 십일조 / 감사 / 선교 / 특별 / 소계
- 절기헌금: 신년 / 부활절 / 맥추절 / 추수 / 성탄 / 소계
- 총수입(= 정기 소계 + 절기 소계)

### 지출 구조 (대분류 = "관")
- 예배비: 사례비 / 성찬비 / 강사 사례비 / 목회비 / 소계
- 선교비: 해외선교 / 국내선교 / 전도비 / 심방비 / 경조사비 / 연금비 / 새가족 / 총회비 / 후원 / 소계
- 교육비: 어린이부 / 청년청소년부 / 신자교육비 / 성가대 / 자녀교육비 / 행사비 / 소계
- **관리비: 월세 / 교회전기 / 인터넷 / 전화 / 교회수도 / 정수기 / 연료비 / 주방 / 자동차 / 사택관리 / 사무비 / 수리비 / 비품비 / 예비비 / 소계**
- 총지출 / 일반지출 / 건축지출

## 수입재정집계표 — 컬럼
`월 | 주 | 주정헌금 | 십일조 | 감사헌금 | 선교헌금 | 절기헌금 | 합계(건축헌금외) | 건축헌금 | 비고`
- 주(週)별 한 행, 월 총계, 맨 아래 총계
- 절기헌금 = 신년+부활절+맥추절+추수+성탄(앱 type 7~11)
- 합계(건축외) = 주정+십일조+감사+선교+절기, 건축헌금 = 앱 type 5

## ⚠️ 핵심 발견 — 지출 종류 누락
앱 `EXPENSE_TYPES`(constants.ts, 19개: 사례비~행사비)에는 **관리비 대분류 항목 전체가 없음**.
→ 재정집계표의 지출 절반(관리비)을 매핑할 대상이 앱에 존재하지 않음.
→ 사용자가 언급한 "지출 항목에 관(管) 부분도 있어"가 이것.
