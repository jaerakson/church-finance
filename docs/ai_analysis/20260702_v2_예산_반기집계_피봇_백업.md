## 📌 TL;DR
- 재정집계표를 반기(상1~6/하7~12) 결산 구조로 재설계: 예산·직전누계·현누계·총합계 자동 계산, 관 접기/펼치기, 금액 클릭 드릴다운, 예산 인라인 입력.
- 연간 예산 관리 페이지 신설 + 구글시트 `예산` 탭 자동 생성(항목별 연 예산).
- 데이터 전체 백업(JSON) 다운로드 + 목록 검색결과 금액 합계 추가.

## ⚠️ 가정 및 한계
- "1년 2번" = 반기 결산으로 해석(사용자 확정). 직전누계=같은 해 이전 반기 누계, 상반기는 직전누계 0.
- 예산 = 항목별 연간 1개(사용자 확정). 총합계(연 누계)와 비교.
- 예산 저장: 같은 스프레드시트에 `예산` 탭(A=연도,B=구분,C=코드키,D=예산액) 자동 생성. 되돌리려면 탭 삭제.
- 페이징/검색은 클라이언트 사이드(수천 건 규모 적합).
- 검증 중 예산 쓰기 테스트는 가상 연도 2099로 수행 후 0으로 리셋 — `예산` 탭에 `2099/income/1/0` 무해한 잔여행 1개 존재(모든 실제 화면에서 안 보임).
