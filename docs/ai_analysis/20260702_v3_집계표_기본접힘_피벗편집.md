## ⚠️ 한계
- 그리드 셀 수정/추가의 실 쓰기(POST/PUT)는 실제 헌금·지출 시트에 기록되므로 라이브 write 테스트는 생략. 다만 기존 입력폼이 쓰는 동일 API를 호출하고, 셀 렌더·값 프리필·커밋 로직은 검증함.
- 같은 (이름,날짜) 2건 이상 셀은 안전상 합계 표시만(수정은 기본 보기). 셀을 비워 삭제하는 기능은 미제공(삭제는 기본 보기에서 확인 후).
