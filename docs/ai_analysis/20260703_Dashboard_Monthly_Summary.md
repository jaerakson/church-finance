## 📌 TL;DR
- 대시보드(재정현황 메인 페이지 `/`) 내에 수입과 지출 현황을 **재정집계표(관별 집계) 형태**로 일목요연하게 파악할 수 있는 신규 영역을 개발하였습니다.
- 사용자가 원하는 월을 손쉽게 고를 수 있도록 최근 월 순서로 정렬된 **월 선택 셀렉트 박스(MonthFilter)** 필터 컴포넌트를 적용하고 연동을 마쳤습니다.

## 구현 상세

### ① 월별 셀렉트 필터 (`MonthFilter`) 추가
- **위치**: `src/components/ui/MonthFilter.tsx` (신규 생성)
- **특징**: 구글 시트 데이터 원장에 존재하는 모든 년-월(예: `2026-06`, `2026-05`)을 자동으로 추출하여 내림차순(최근 순)으로 셀렉트 박스 옵션을 빌드합니다.
- **연동 흐름**: 월을 선택하면 URL 쿼리 파라미터(`?month=YYYY-MM`)가 갱신되고 연도 필터는 상충되지 않도록 자동 제어됩니다.

### ② 관(대분류)별 재정집계표 구현
- 대시보드 상단에 **'정기헌금', '절기헌금', '예배비', '선교비', '교육비', '관리비', '건축비'** 등 핵심 관 분류별로 당월 수입 실적과 지출 실적, 그리고 최종 잔액(차액)을 비교해 주는 집계 테이블을 삽입했습니다.
- **실적 발생 항목 필터링**: 가독성을 극대화하기 위해 당월 실적이 한 푼이라도 기록된 유의미한 항목만 집계 표에 노출하고, 하단에 총합계를 정밀 계산해 줍니다.
- **실시간 요약 카드 동기화**: 상단 요약 카드(당월 헌금액, 당월 지출액, 잔액) 역시 선택된 월에 동적으로 실시간 연동되어 완벽한 싱크로율을 보장합니다.
